# خطاها

<aside class="notice">
خطاهای زیر می‌توانند برای هر یک از انواع درخواست‌ها دریافت شوند. اکثر خطاها شامل یک
JSON
به همراه <code>message</code> و یا <code>messages</code> می باشند.
</aside>

API جواب‌کو شامل کدهای خطای زیر می‌باشد:

  <thead>
